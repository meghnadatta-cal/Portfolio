YouTube Clone - React Video Streaming Platform

The app integrates the YouTube Data API v3 to fetch and display real-time video content, including metadata, thumbnails, and related recommendations. Implemented dynamic routing with react-router-dom for seamless navigation between home and individual video pages. The project showcases practical use of JSX, props, async/await, event handling, and third-party libraries like Moment.js for date formatting.
